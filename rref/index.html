Rvalue references for G++

Hi, this is a little project page for my attempt to add rvalue references to G++. Eventually, I should have a complete patch that I'll submit for inclusion in GCC, but for now I'm just working off the GCC 4.0.0 sources and tracking my changes in a Subversion repository. The address of the repository and a copy of the latest diff are available below.

Russell Yanofsky
russell.yanofsky@us.army.mil

Download

My changes are available from a public Subversion repository at http://russ.hn.org/svn/rref/

If you don't have Subversion, you can get a cached copy of the latest diff here or browse the repository with ViewCVS

If you do have Subversion, you can check out the sources with

svn co http://russ.hn.org/svn/rref/trunk/ rref

or get the latest diff with

svn diff http://russ.hn.org/svn/rref/vendor/gcc/4.0.0 http://russ.hn.org/svn/rref/trunk
