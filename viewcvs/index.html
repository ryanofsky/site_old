ViewCVS 1.0-dev

The Windows and Unix versions of ViewCVS have finally been united! The next official version of ViewCVS will include all of the functionality that was in previous Windows patches. In the meantime, though, I've put up a new release that includes all of the ViewCVS improvements that have been made over the last couple of months, a number of important windows-specific bugfixes, a more comprehensive readme, and support for mod_python for better performance under Apache.

You can send bug reports or questions to me at rey4@columbia.edu, but please also forward them to the viewcvs mailing list at viewcvs@lyra.org so other people can benefit from the information.

Thanks,

Russ Yanofsky

Releases

  • ViewCVS 1.0-dev for Windows Release 3, February 18, 2003. Merged with latest development version of ViewCVS, bugfixes for Apache, recent versions of CVSNT, and for the commit database. Added support for mod_python.
  • ViewCVS 1.0-dev for Windows Release 2, July 8, 2002. Minor bugfixes and updated readme.
  • ViewCVS 1.0-dev for Windows Release 1, March 06, 2002. Full port of ViewCVS to Windows. New "sapi" module to allow ViewCVS to work under ASP.
  • ViewCVS 0.7 for Windows, July 22, 2001. Converted ViewCVS code to work with Windows line endings and the IIS CGI environment. No support for checkin database.
