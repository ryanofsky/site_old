ViewCVS 1.0-dev

On this page you can download the latest and greatest version of ViewCVS for Windows. Unlike the first release, this one supports all of the ViewCVS features available on unix including CvsGraph diagrams, tarball download links, bonsai commit database queries, regular expression filtering, code highlighting, and use of the viewcvs standalone server.

The biggest improvement, though, is that ViewCVS can now run under ASP instead of CGI, giving it much better performance. The ASP support is transparent so you can run ASP or CGI or both at once, without any changes to the source code or configuration.

You can send bug reports or questions to me at rey4@columbia.edu, but please also forward them to the viewcvs mailing list at viewcvs@lyra.org so other people can benefit from the information.

Thanks,

Russ Yanofsky
