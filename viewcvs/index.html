ViewCVS for Windows

ViewCVS is a web application used by developers to browse their favorite CVS repositories. So far there is no official release of ViewCVS that runs on Microsoft Windows, so I've created this page to make unofficial versions that do work more readily available. Recently, the changes needed to support Windows have been integrated into the ViewCVS development sources and the newer downloads here are just snapshots of those sources. If you have questions or need technical support for any version of ViewCVS, post to the ViewCVS mailing list (or its NNTP interface).

Releases

  • ViewCVS 1.0-dev for Windows Release 4, June 8, 2003. More bugfixes and updated readme.
  • ViewCVS 1.0-dev for Windows Release 3, February 18, 2003. Merged with latest development version of ViewCVS, bugfixes for Apache, recent versions of CVSNT, and for the commit database. Added support for mod_python.
  • ViewCVS 1.0-dev for Windows Release 2, July 8, 2002. Minor bugfixes and updated readme.
  • ViewCVS 1.0-dev for Windows Release 1, March 06, 2002. Full port of ViewCVS to Windows. New "sapi" module to allow ViewCVS to work under ASP.
  • ViewCVS 0.7 for Windows, July 22, 2001. Converted ViewCVS code to work with Windows line endings and the IIS CGI environment. No support for checkin database.
